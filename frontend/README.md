# Badminton Court Booking System - Frontend

This is the frontend for the Badminton Court Booking System. It is built with React and Vite.

## Setup Instructions

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Access the application:**
    Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Features

*   **Book a Court:** Select date, time, court, equipment, and coach.
*   **Live Pricing:** See the price breakdown update instantly as you select options.
*   **Admin Dashboard:** Toggle availability of courts and view pricing rules/equipment.

## Tech Stack

*   **React** (Vite)
*   **CSS Modules** (via vanilla CSS variables for theming)
*   **Axios** for API communication
*   **Date-fns** for date handling
*   **React Router** for navigation
